# Linux Mini Projects

This repository contains hands-on **Linux mini projects** designed to build deep practical understanding of system administration concepts.  
Each project is **fully scripted**, and includes a detailed `README.md` explaining the logic, commands, and real outputs.

---

## Current Projects

| Project | Description | Path |
|----------|--------------|------|
| **Partition Management** | Automates disk partitioning using `parted`, formatting, mounting, and persistent fstab configuration. | [partition-management](./partition-management) |
|  **Swap Management** | Creates and manages swap files, adds additional swap, verifies kernel usage, and handles cleanup with persistence. | [swap-management](./swap-management) |
| **LVM Management** | Builds full LVM stack (PV → VG → LV), performs extensions, verifies mount persistence, and supports automated cleanup. | [lvm-management](./lvm-management) |
